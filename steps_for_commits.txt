0. устанавливаем Git на компьютер
1. создаем папку нашего нового проекта
2. запускаем git bash
3. в командной строке переходим в созданную папку, команда cd
4. инициализируем репозиторий - находясь в папке проекта, пишем git init
теперь все файлы, которые будут в нашей папке, будут отслеживатья by Git
5. создаем файлы в папке нашего проекта - файлы с кодом, комментарии и т.д. (я даже копирнул туда 40мегабайтный файл hdf5)
6. добавляем файлы для будущего коммита командой git add:
git add myfile.txt - по одному файлу
git add --all - все сразу файлы в папке
git add . - всю текущую папку
7. проверяем что у нас добавилось для коммита командой git status
8. делаем коммит командой git commit -m 'тут комментарий'. Обязательно пишем комментарий к коммиту.
историю коммитов смотрим командой git log
9. закидываем наш коммит (он пока локальный) на гитхаб командой gitpush
10. заходим на гитхаб, открываем нужный репозиторий и проверяем, что наш коммит появился тут (ну и файлы из нашего проекта все конечно забросились на гитхаб)
